# UTEHub Frontend

React.js frontend application for UTEHub social network.

## Setup

```bash
cd frontend
npm install
npm run dev
```

## Project Structure

```
frontend/
├── src/
│   ├── components/     # Reusable components
│   ├── pages/          # Page components
│   ├── redux/          # Redux store, slices
│   ├── services/       # API services
│   ├── styles/         # Tailwind CSS
│   ├── utils/          # Helper functions
│   ├── App.jsx
│   └── main.jsx
├── public/
├── package.json
└── vite.config.js
```

## Features

- User authentication
- Feed with posts
- Comments and likes
- Real-time notifications
- Friend search and management
- Community/Group management
